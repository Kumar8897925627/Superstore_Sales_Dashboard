# Superstore_Sales_Dashboard
A Power BI dashboard showcasing sales performance across different regions, categories, and time periods.
Superstore Sales Dashboard

A Power BI dashboard visualizing sales data from the Superstore Sales dataset.

Description:

This repository contains a Power BI dashboard that provides insights into sales data from the Superstore Sales dataset. The dashboard includes three visuals:

The dashboard also includes a slicer for Region and highlights top-performing areas using colors.

Features:

- Interactive visuals for drill-down analysis
- Slicer for filtering data by Region
- Color-coded highlights for top-performing areas
- Insights into sales trends and performance by Region and Category

Data Source:

- Superstore Sales dataset (CSV file)

Tools Used:

- Power BI

How to Use:

1. Clone the repository and open the Power BI file (.pbix)
2. Load the Superstore Sales dataset (CSV file)
3. Interact with the visuals and slicer to analyze the data
4. Customize the dashboard as desired

Insights:

- West region had the highest sales in Q4, driven by strong performance in the Technology category.
- The Furniture category saw a significant increase in sales in the South region, up 15% YoY.
- The Central region is the lowest-performing region, with sales down 5% YoY, primarily due to a decline in the Office Supplies category.
